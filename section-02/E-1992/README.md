## 쿼드 트리

### 제한
- 시간 : 2초
- 메모리 : 128MB

### 문제 의도
- 한 줄씩 입력을 받아서, 원하는 배열에 담아 정보를 처리

### 입력
```
8   // 영상 크기, 2의 제곱수, 2^0 <= N <=2^5
11110000    // 1 : 검은 점, 0 : 흰 점
11110000
00011100
00011100
11110000
11110000
11110011
11110011
```

- 고려 사항
2^0 -> 그대로 출력
2^1 -> 상, 하, 좌, 우가 같으면 그대로 출력 / 아니면 순서대로 출력
2^2 -> 2^1 크기로 분할
2^3 -> 2^2, 2^1, 2^0 크기대로 분할 후 검사 

```
11 11   00 00
11 11   00 00

00 01   11 00
00 01   11 00


11 11   00 00
11 11   00 00

11 11   00 11
11 11   00 11
```

- 2^n -> 최하위 그룹의 수
- 2로 나눠가면서 정보를 요약
- 첫 줄 그룹의 수 : 1, 1, 2, 4, 8 .. -> 2^(n-1)
- 2n -> 새로운 그룹에 담음 / 2n - 1 -> 이전 그룹에 담음
