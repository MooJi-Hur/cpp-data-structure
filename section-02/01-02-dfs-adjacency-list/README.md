### 입력 그래프
```
     A
    / \
   B   C
  / \  /
 D   E
  \ /
   F
```

### 인접 리스트
- A : B, C
- B : A, D, E
- C : A, E
- D : B, F
- E : B, C, F
- F : D, E 

### DFS 탐색 규칙
1. 시작 노드를 스택에 넣는다. 
2. 시작 노드부터 인접 리스트를 조회한다. 
3. 이전에 방문한 적이 없는 노드인 경우, 스택에 넣는다. 
4. 인접리스트를 조회한 큐는 제거한다. 
5. 다음 스택 원소를 꺼내어 1 ~ 4를 반복한다. 
6. 스택이 비어있으면 중단한다.
